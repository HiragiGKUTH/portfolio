# BeautifyUnkode

web application for my class

## What's this

This application beautify your "Unkode" (means shit code)

example:

```cpp:shit.c
#include <stdio.h>

int main() {
printf("hoge");
return 0;
}
```

to

```cpp:beauty.c
#include <stdio.h>

int main() {
    printf("hoge");
    return 0;
}
```
